فرانت اند چیست؟

فرانت‌اند و بک‌اند دو بخش اصلی یک وب‌سایت یا نرم‌افزار هستند. فرانت‌اند همان چیزی است که کاربر می‌بیند و با آن تعامل دارد؛ مثل طراحی صفحات، دکمه‌ها، منوها و رنگ‌ها. فناوری‌های رایج آن شامل HTML، CSS و JavaScript است. در مقابل، بک‌اند پشت صحنه قرار دارد و وظیفه‌اش مدیریت داده‌ها، ارتباط با پایگاه داده و اجرای منطق برنامه است؛ زبان‌هایی مثل Python، PHP، Java و فریم‌ورک‌هایی مانند Django یا Node.js در این بخش استفاده می‌شوند. به طور خلاصه، فرانت‌اند ظاهر و تجربه کاربری را می‌سازد و بک‌اند عملکرد و پردازش داده‌ها را مدیریت می‌کند. این دو بخش برای ساخت یک وب‌سایت کامل به صورت هماهنگ کار می‌کنند.
